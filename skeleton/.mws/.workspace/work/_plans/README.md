# work/_plans/

Inbox for Claude Code plan-mode artifacts. Targeted by `plansDirectory` in `.claude/settings.json`, so every plan generated in plan mode lands here with a random slug (`abundant-mapping-seahorse.md`).

These files are ephemeral by design:

- **Implemented** -- delete the file once the plan ships.
- **Promoted to a named initiative** -- copy into `work/<slug>/plan/` so it lives next to that initiative's research, notes, and artifacts. Then delete the inbox copy.
- **Superseded** -- delete (or move to `work/_archive/_plans/` if you want history).

If you find yourself protecting a plan here for more than a sprint, that's the signal to promote it into a real `work/<slug>/`.
