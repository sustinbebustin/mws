# .workspace

AI-assisted development workspace for the meta workspace. See [INDEX.md](./INDEX.md) for the full map.

Lifecycle-first layout:

- `knowledge/` -- durable, curated; agents read this
- `work/` -- per-initiative artifacts (plans, research, notes); archive when shipped
- `lab/` -- executable exploration scripts and captured runs
- `tickets/` -- issue tracker mirror
- `decisions/` -- lightweight ADRs
- `journal/` -- session/day notes
