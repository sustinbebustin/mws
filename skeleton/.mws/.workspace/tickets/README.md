# tickets/

Jira mirror. Owned by the `jira` skill (`.claude/skills/jira/`).

- `jira-latest.csv` -- the cached CSV export the skill diffs against
- `sprints/` -- per-sprint markdown (`<YYYY-MM-DD>-sprint.md`)
- `cache/` -- scratch space for the skill if needed

Don't edit these by hand if you're going to re-run the skill -- it will overwrite metadata.
